Creating a new number system from scratch is a fantastic exercise in logic and creativity! Most of the world uses
Base-10 (Decimal) because we have ten fingers, but we can build a system based on anything.

Let's create a new, highly visual number system called **Poly-6**.

### The Core Concept: Poly-6

Poly-6 is a **Base-6 (Senary) positional number system**. It relies entirely on geometric shapes. Instead of having to
memorize arbitrary squiggles like "3" or "8," the value of a single digit in Poly-6 is exactly equal to the number of
points or edges it has.

Because it is Base-6, we only need symbols for the numbers 0 through 5. Once we hit 6, we carry over to the next "place
value" (just like we do in Base-10 when we hit 10).

### The Digits (Symbols)

| Decimal Value | Poly-6 Symbol | Name     | Logic / Shape                                         |
|---------------|---------------|----------|-------------------------------------------------------|
| **0**         | **◯**         | Null     | A smooth, empty circle (no points or straight edges). |
| **1**         | **•**         | Point    | A single dot.                                         |
| **2**         | **            | **       | Line                                                  | A straight line (connecting two points). |
| **3**         | **△**         | Trigon   | A triangle (3 edges).                                 |
| **4**         | **□**         | Tetragon | A square (4 edges).                                   |
| **5**         | **⬠**         | Pentagon | A pentagon (5 edges).                                 |

---

### How to Count in Poly-6

In our standard Decimal system, the place values represent 1s, 10s, 100s, 1000s, etc.
In Poly-6, the place values represent **1s, 6s, 36s, 216s**, etc.

Let's count from 0 to 12 to see how the numbers carry over:

* **0:** ◯
* **1:** •
* **2:** |
* **3:** △
* **4:** □
* **5:** ⬠
* **6:** •◯ *(One "six" and zero "ones" — similar to how 10 works in decimal)*
* **7:** •• *(One "six" and one "one")*
* **8:** •| *(One "six" and two "ones")*
* **9:** •△
* **10:** •□
* **11:** •⬠
* **12:** |◯ *(Two "sixes" and zero "ones")*

---

### Basic Math in Poly-6

The beauty of this system is that it is visually intuitive. If an alien or a young child looked at it, they could
physically count the points/edges to do the math.

**Addition Example:**
Let's add 3 and 2 in standard Decimal: $3 + 2 = 5$.
In Poly-6, this looks like: **△ + | = ⬠** *(3 edges + 2 points/edges = 5 edges)*.

**Carrying Over Example:**
Let's add 4 and 3 in standard Decimal: $4 + 3 = 7$.
In Poly-6, a square (4) plus a triangle (3) gives you 7 total edges. But we only have symbols up to 5! So, we bundle 6
of those edges into a new place value, and keep the 1 leftover edge.
**□ + △ = ••** *(One "six" + one "one" = 7)*.

---

### Why use a Base-6 system?

Base-6 is actually mathematically fantastic. The number 6 is highly divisible. It can be divided cleanly by 1, 2, 3, and
6. This makes fractions incredibly easy to work with (for example, $1/3$ in Base-6 is simply represented as $0.2$,
because 2 is exactly one-third of 6).
