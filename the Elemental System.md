Here is a brand new, custom-designed number system. Let's call it the **Elemental System**.

Instead of relying on ten digits like our standard decimal system, this system operates on a **Base-5 (Quinary) structure**, using visual symbols inspired by nature to represent values.

### 1. The Digits (0 through 4)

Because it is a Base-5 system, we only need five symbols. Instead of arbitrary shapes, the symbols visually build on the concept of natural elements:

| Value (Base-10) | Symbol | Element Name | Visual Logic |
| --- | --- | --- | --- |
| **0** | `O` | Void | An empty circle, representing nothingness. |
| **1** | `~` | Water | A single wave. |
| **2** | `=` | Earth | Two layers of sedimentary rock. |
| **3** | `^` | Fire | A triangle representing the three points of a flame. |
| **4** | `+` | Air | A cross representing the four winds/directions. |

### 2. How Place Value Works

In our normal Base-10 system, each column is a power of 10 (1s, 10s, 100s). In the Elemental System, each column to the left is a **power of 5**.

* **1st position (right-most):** The "Ones" column ($5^0$)
* **2nd position:** The "Fives" column ($5^1$)
* **3rd position:** The "Twenty-fives" column ($5^2$)
* **4th position:** The "One-hundred-twenty-fives" column ($5^3$)

### 3. Examples in Action

To read or write a number, you figure out how many 25s, 5s, and 1s fit into it, and then write the corresponding symbols from left to right.

**Example 1: The number 7**

* How many 5s? One. (Symbol: `~`)
* How many 1s leftover? Two. (Symbol: `=`)
* **Elemental Translation:** `~ =`

**Example 2: The number 14**

* How many 5s? Two. (Symbol: `=`)
* How many 1s leftover? Four. (Symbol: `+`)
* **Elemental Translation:** `= +`

**Example 3: The number 25**

* How many 25s? One. (Symbol: `~`)
* How many 5s? Zero. (Symbol: `O`)
* How many 1s? Zero. (Symbol: `O`)
* **Elemental Translation:** `~ O O`
